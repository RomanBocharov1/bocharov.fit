Перехват http: (изначально в постмане доступен олько перехват http)
- capture request - галочка on -активируется proxy
- браузер - настройки - поиск по слову proxy\ настройки proxy
- выставляем адрес http:\\localhost и порт из постмана
- делаем запрос на http и смотрим результат в history

Комментарий:
-В постмане мы можем видоизменять запрос для тестов
- interceptor postman - расширение для хрома, чтобы постман мог захватывать https
- можем изменить запрос на сервер и посмотреть , как тот ответит(для тестирования)
Front-back:
При каком-то событии происходит запрос (фронт прописывает запрос(html страница допустим) - запрос идет на бэк и подтягивается оттуда ресурс)
http:\\localhost - это доменное имя ПК
http:\\127.0.0.1\ это ip пк
мой пк - это виртуальный сервер
http:\\192.168.х.х\ - доступ к серверу с отдельного компьютера в той же сети
Когда прописываем пути: / значит зайти в папку на сервере , .. выйти из папки на сервере
