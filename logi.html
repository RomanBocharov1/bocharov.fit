Логи - документация, где содержится информация о событиях и времени, когда это произошло, ошибках, статистике. Существуют: Логи приложения - (браузер, мобильное приложение) Логи сервера - error logs: информация об ошибках и спец.инфа - access logs: общая информация о всех запросах Системные логи Уровни логирования: (указываются при запуске программы, зависти от того, что хотим посмотреть, т.к если будут выводиться все, то это слишком много информации) off - не записываются fatal - блокирующая систему ошибка error - ошибка, но система работает warn - предупреждение info - служебная информация(создания, удаления) debug - отладка(включает все вышеперечисленное) trace - все остальное all - все Струтура лога: ip\ время\ тип запроса\ http версия\ код состояния\ ОС\ браузер Комментарий: ELK - elastic search logstash (инструмент для анализа логов)
