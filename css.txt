CSS(вид html страницы(тегов HTML): шрифт,отступ,анимация т.д.)
таблица каскадных стилей(водопад, наследие) , т.е. идет наследование(отцы-дети)
можно применть к любому тэгу: style = ""
если применяем к body или первому тегу HTML(наверху страницы), то применяем ко всей странице.
если применяем к тэгу внутри body, то этот тэг меняется, но body остается прежним.
style="color:red" (цвет)
<h2 style = "color:red"> .... </h2>
<h2 style = "color:red";margin="1px"> .... </h2> (через ; перечисляются свойства)

<link rel = "stylesheet" href = "bilan.css"(подключение css файла, название всегда по английски)
(подключать можно бесконечное кол-во файлов) в href можно вставить ссылку на css фреймворк

---------
в css файле: 
table         (селектор - то, для чего мы меняем стиль)
{
color:blue;  (свойство: значение)
border:2px;  (свойство: значение)
}
-----------
можем взять любой атрибут и дать ему стиль(но нужны квадратные скобки)
[href="hhh.jpg"] {
color:red;
}
-------------
если поставить тэг впереди,стиль будет действовать для любого тэга с этим атрибутом
li[href="hhh.jpg"] {  
color:red;
}
-------------------
class(контейнер. то что обьединяет) используется .class внутри css, чтобы связать стиль 
с тэгом внутри html документа
в css:
.table-red{
color:red;
}
.table-bold{
font-weght:bold;
}
в html документе: 
<table class = "table-red table-bold"> .... </table>  (соответственно, текст в таблице 
будет красным и жирным)  
можно добавлять бесконечное количество классов и к любым тэгам много раз

--------------------------------
в css:
#table-red{
color:red;
}

#table-bold{
font-weght:bold;
}
в html документе: 
<table id = "table-red table-bold"> .... </table>  
(используется один раз, в отличии от класса)
-----------------------------
html tr {   
color:red
}
(название документа и через пробел тег- стиль применяется ко всем этим тэгам внутри документа html)
вместо тэга можно использовать атрибут , id ,class
-------------
.list > li{   
color:red
}
(стиль будем применяться только к 1 уровню вложенности, если убрать > красным будет все, потомки тоже)
