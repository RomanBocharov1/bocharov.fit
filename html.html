<!doctype HTML> (документ HTML 5-ой версии)

<HTML> (открытый тэг)
<head>
<title> название вкладки </title>
<meta charset="utf-8"> (кодировка, чтобы русский язык был читаем)
<meta name = "keywords" content = "мой сайт"> (для индексации)
<meta name = "discription" content = "описание моего сайта">(для индексации)
<link rel = "icon" href= "...ico"/> (иконка вкладки)
</head>
<body>
 <h1> заголовок </h1> (2,3,4 - уменьшаются по размеру, занимают всю строку)
<p> абзац</p>
<br> (перенос строки)
<b> жирный текст </b>
<em> курсив </em>
<a href = "ссылка"> текст\ картинка с ссылкой</a>
<a href = "ссылка" target="blank">текст\ картинка с ссылкой откроется в новой вкладке</a>
<p id = "1"> текст внутри</p>
<a href = "#1"ссылка на строку выше </a>
<ul>
<li>первый элемент списка</li>
<li>второй элемент списка</li>
</ul>
<hr> (делаем разделительную линию)
<img crs = "c:/users ..... " width="50" height="50">
<input type = "text" > (поле для введения текста)
<input type = "number"> (поле для введения чисел)
<input type = "month"> (выбор месяца)
<div class="....">.......</div> (блочный элемент)

</body> (закрывающий тэг)
 </HTML> (закрывающий тэг)
Комментарий:
<>.......</> закрытый тэг
<> открытый тэг(не требует закрытия)
head не отображается на странице
href - hyper reference
Чтобы отобразить код в браузере на HTML странице, вместо < и > пишем & lt ; и & gt ;(без пробелов)
